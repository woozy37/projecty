v0.0.3 (En développement)

- Population d'arbre et de plantes random sur la map en détectant la collision des reliefs du terrain.
- Un personnage
- Caméra qui suit le personnage + controles caméra + controles personnage
- Optimisations textures
- Interface GUI (barre de vie ou autre)
- Implémentation de la lumière réaliste sur la map.

La v0.0.3 est un grand patch pour le moteur de jeu java qui permettra déja d'obtenir des résultats suffisant pour un gameplay Mais comme je l'avais annoncé, la partie jeu ne m'interresse que très peu finalement. Je veux pouvoir concevoir un monde optimisé, sans règles pour le moment.

Bienvenue sur Island Game Project

Je posterai ici l'avancée d'un jeu 3d développé avec Java et OpenGl, et la librairie LWJGL.

Les modèles 3d seront conçus avec Blender.

Le but du projet

Dans un premier temps, je ne compte pas faire de gameplay ni de règles du jeu. Il s'agit d'un entrainement complet, pour ce qui s'avère être ma première approche du jeu vidéo. Je compte d'abord développer une île, un personnage, les annimations de celui ci sur son environnement, et bien sur, le moteur graphique pour permettre tout ça.

Création d'un auteur (nbdy). Source : http://nbdy.deviantart.com/art/3D-Island-256309881

La suite de l'aventure

Dans un deuxième temps, j'essairai de faire evoluer ce projet en un jeu completement jouable. Mais ce n'est pas encore d'actualité.
